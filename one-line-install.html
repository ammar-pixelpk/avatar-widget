ðŸŽ¯ Customization Options

Customize the widget by changing the data attributes:

<script src="https://cdn.jsdelivr.net/gh/your-username/avatar-widget@latest/installer.js" 
        data-knowledge-id="YOUR_KNOWLEDGE_ID"
        data-avatar-id="Anna_public_3_20240108"
        data-opening-message="Welcome! I'm Anna, how can I assist you?"
        data-position="bottom-left"
        data-theme="dark"></script>

Available Options:

  • data-knowledge-id: Your knowledge base ID (required)
  • data-avatar-id: Avatar to use (Tyler, Anna, Susan, etc.)
  • data-opening-message: Initial greeting
  • data-position: bottom-right, bottom-left, top-right, top-left
  • data-theme: light, dark
  • data-size: small, medium, large
